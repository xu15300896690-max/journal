## 🚀 使用建议

### 首次使用
1. 启动 Windows 服务器
2. 配置 Android 服务器地址
3. 用 admin/admin123 登录
4. 添加基础数据（供应商/客户/物品）
5. 开始使用入库/出库功能

### 日常使用
1. 入库时拍照记录车牌
2. 出库时扫码快速录入
3. 定期查看库存预警
4. 每月导出报表统计
5. 需要时打印单据

### 维护建议
1. 定期备份数据库
2. 清理过期数据
3. 更新物品最低库存
4. 检查系统日志
